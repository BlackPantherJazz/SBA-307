# Henny Music Website

## Description

A 3-page music website for artist Henny. The site showcases music, visuals, and booking options.

## Pages

* Home
* Music
* Contact

## Features

* Table of songs
* Two forms (contact + booking)
* Dropdown menu
* Images and GIFs
* Animated red gradient background
* Flexbox navigation

## Technologies Used

* HTML
* CSS
